팀 프로젝트 협업

이 프로젝트는 HTML/CSS/JS 기반의 다양한 보안 이슈 소개 페이지를 주제로 진행되었습니다. 단순한 코딩 기술의 합을 넘어, 팀원 간 조화로운 의사소통과 각자의 강점을 결합하여 완성도 높은 반응형 사이트를 완성했습니다.

기획 단계에서 레이아웃 아키텍처를 잡고, GitHub 브랜치 전략을 적극적으로 활용하여 코드 충돌을 방지하며 사용자가 쉽게 여러가지 보안 이슈를 이해할 수 있도록 고민하며 사이트를 완성했습니다.
